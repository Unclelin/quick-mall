# Quick Mall

##平台介绍

  这是一个快速搭建在线商城的平台，还在努力开开发中敬请期待 :)

## 所以技术

1. spring boot
2. shiro
3. redis
4. swagger2
5. shareding-jdbc
6. mysql
7. vue /react
8. adminLET
9. activeMQ/RocketMQ
10. Ueditor
11. Echart
12. nginx

## 访问URL

http://localhost:9081/admin/test/tom

hello world Test地址:
